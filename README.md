# Agentic AI Explorer

This repository contains an interactive guide to Agentic AI.

The main content is in the `index.html` file. Open this file in a web browser to view the guide.

The guide explores:
- The basics of AI Agents (Perception, Planning, Action, Memory).
- An interactive "Agentic Brainstormer" powered by the Gemini API.
- The evolution of agentic concepts (Chain-of-Thought, ReAct Framework, Autonomous Agents).
- A filterable timeline of major Large Language Model releases.
- A filterable grid of AI-native IDEs and development tools.

The page is designed to be a single-page, thematic scrolling experience with a sticky top navigation and includes a dark/light theme toggle.
It uses Tailwind CSS for styling and Chart.js for visualizations.
